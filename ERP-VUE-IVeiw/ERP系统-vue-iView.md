## vue-cli 讲解链接 https://my.oschina.net/liuyuantao/blog/883251 ##
- vue-cli  Vue官方推荐
  + 安装vue-cli  
    - npm install vue-cli -g
  + 初始化项目  
    - vue init webpack vuecliTest 
  + 安装依赖包
    - npm instal
  + 运行
    - npm run dev
